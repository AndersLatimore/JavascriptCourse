Slide Up Down: A jquery plugin

This plugin slides the image up and down when clicked.

All you need to do to use the plugin is download it here and the define the elements as seen below. #box10 is the div where you want to use the plugin. example is where you define the image that you want to use the the plugin on. You can set the speed of the animation by changing the duration parameter.

$('#box10 img.example').click(function() { $(this).slideUpDown({'duration':5000}); console.log('Clicked image to slide it up and down.'); });
